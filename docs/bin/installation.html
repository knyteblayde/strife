Installation

Installations are always a headache at first if you're not-so-savvy programmer, so to avoid being cursed for a week
figuring out how to use a certain tool, we provided three easy options on how to install spiral framework:

1. First option is to download the package
head to the Download page and pick the latest and stable version, extract the contents inside your web accessible directory on your web server or your local development server like Xampp for example is usually C:\xampp\htdocs while on Wampserver it is inside C:\wamp\var\www.

2. Second Option is to clone the repository
Spiral Framework is regularly updated on a repository on Github, you can get the latest build or the lower versions of it by cloning the repository by issuing the command below (assuming you have Git pre-installed on your computer; if not, google for it): 

git clone https://github.com/knyteblayde/spiral.git

3. Third and Last option is to use composer
Composer is a powerful php dependency manager, it intelligently handles the autoloading of your classes and application components.
If you have no idea about composer, please make sure to visit getcomposer.org on how to get it work.

We will use composer to pull in Spiral package: 
composer require "knyteblayde/spiral" : "~dev-master"
